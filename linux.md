# Basic commands for Linux

## File
|command|description|
|-------|-----------|
|ln -s <source> <target>|Create a symbolic link|

## Process
|command|description|
|-------|-----------|
|ps -aux|Show all processes|
|kill \<pid\>|Kill a process|

## Network
|command|description|
|-------|-----------|
|netstat -tulpn|Show all open ports|
